# 输出模板和示例

## 模板系统

### 1. 时间模板 (time)
**格式**：`当前时间是 {time_24h}`

**变量**：
- `{time_24h}`: 24小时制时间 (HH:MM:SS)
- `{time_12h}`: 12小时制时间 (hh:MM:SS AM/PM)

**示例**：
- 中文：当前时间是 21:50:15
- 英文：The current time is 09:50:15 PM

### 2. 星期模板 (weekday)
**格式**：`今天是{weekday_zh}`

**变量**：
- `{weekday_zh}`: 中文星期
- `{weekday_en}`: 英文星期

**示例**：
- 中文：今天是星期四
- 英文：Today is Thursday

### 3. 日期模板 (date)
**格式**：`今天是{date_zh} {weekday_zh}`

**变量**：
- `{date_zh}`: 中文日期 (YYYY年MM月DD日)
- `{date_ymd}`: 标准日期 (YYYY-MM-DD)
- `{weekday_zh}`: 中文星期

**示例**：
- 中文：今天是2026年03月19日 星期四
- 英文：Today is 2026-03-19 Thursday

### 4. 完整信息模板 (full)
**格式**：`当前日期时间是：{full_datetime_zh}`

**变量**：
- `{full_datetime_zh}`: 完整中文日期时间
- `{full_datetime}`: 完整标准日期时间

**示例**：
- 中文：当前日期时间是：2026年03月19日 星期四 21:50:15
- 英文：Current date and time: 2026-03-19 Thursday 21:50:15

### 5. 月份模板 (month)
**格式**：`现在是{month}月`

**变量**：
- `{month}`: 月份数字
- `{month_zh}`: 中文月份名称

**示例**：
- 中文：现在是3月
- 英文：It's March

### 6. 年份模板 (year)
**格式**：`今年是{year}年`

**变量**：
- `{year}`: 年份数字

**示例**：
- 中文：今年是2026年
- 英文：It's 2026

## 错误处理模板

### 1. 时间获取失败
**格式**：`无法获取当前时间，请检查系统时间设置`

**使用场景**：
- datetime.now() 调用失败
- 系统时间异常

### 2. 格式转换失败
**格式**：`当前时间：{default_format}`

**变量**：
- `{default_format}`: 默认格式的时间字符串

**使用场景**：
- 格式化失败时使用简单格式

### 3. 意图识别失败
**格式**：`当前日期时间：{full_datetime_zh}`

**使用场景**：
- 无法识别用户意图时返回完整信息

## 语言适配

### 中文模式（默认）
- 使用中文模板
- 包含中文星期和日期格式
- 使用"年"、"月"、"日"、"星期"等中文单位

### 英文模式（可选）
- 使用英文模板
- 包含英文星期和日期格式
- 使用英文单位和表达

### 语言检测规则
1. 如果用户使用英文提问，考虑使用英文回复
2. 如果用户使用中文提问，使用中文回复
3. 保持会话语言一致性

## 输出美化

### 表情符号
根据时间和意图添加适当的表情符号：

| 意图 | 白天 (06:00-18:00) | 晚上 (18:00-06:00) |
|------|-------------------|-------------------|
| 时间 | ⏰ | 🌙 |
| 日期 | 📅 | 📅 |
| 星期 | 📆 | 📆 |
| 完整 | 📅⏰ | 📅🌙 |

### 格式化建议
1. **加粗重要信息**：`**21:50:15**`
2. **使用列表**：对于多部分信息
3. **保持简洁**：避免冗长描述

## 模板使用指南

### 在脚本中使用
```python
from scripts.get_datetime import format_by_intent

# 获取时间信息
datetime_info = get_current_datetime()

# 根据意图格式化
user_query = "现在几点了？"
intent = detect_intent(user_query)
output = format_by_intent(datetime_info, intent)

# 添加表情符号
if intent == "time":
    if 6 <= datetime_info["hour"] < 18:
        output = f"⏰ {output}"
    else:
        output = f"🌙 {output}"
```
