# 日期时间格式参考

## Python datetime 格式化代码

| 代码 | 含义 | 示例 |
|------|------|------|
| %Y | 4位数的年份 | 2026 |
| %y | 2位数的年份 | 26 |
| %m | 月份（01-12） | 03 |
| %B | 月份全名 | March |
| %b | 月份缩写 | Mar |
| %d | 日（01-31） | 19 |
| %H | 24小时制小时（00-23） | 21 |
| %I | 12小时制小时（01-12） | 09 |
| %M | 分钟（00-59） | 50 |
| %S | 秒（00-59） | 15 |
| %p | AM/PM | PM |
| %A | 星期全名 | Thursday |
| %a | 星期缩写 | Thu |
| %w | 星期数字（0=周日） | 4 |
| %j | 一年中的第几天（001-366） | 078 |
| %U | 一年中的第几周（周日为一周开始） | 11 |
| %W | 一年中的第几周（周一为一周开始） | 11 |

## 常用格式模板

### 中文格式
1. **完整日期时间**：`YYYY年MM月DD日 星期X HH:MM:SS`
   - 示例：2026年03月19日 星期四 21:50:15

2. **日期**：`YYYY年MM月DD日 星期X`
   - 示例：2026年03月19日 星期四

3. **时间**：`HH:MM:SS`
   - 示例：21:50:15

4. **简短日期**：`YYYY-MM-DD`
   - 示例：2026-03-19

### 英文格式
1. **完整日期时间**：`YYYY-MM-DD HH:MM:SS Weekday`
   - 示例：2026-03-19 21:50:15 Thursday

2. **12小时制**：`YYYY-MM-DD hh:MM:SS AM/PM`
   - 示例：2026-03-19 09:50:15 PM

## 输出格式选择规则

### 根据用户意图选择格式

| 用户意图 | 推荐格式 | 示例输出 |
|----------|----------|----------|
| 时间查询 | HH:MM:SS | 当前时间是 21:50:15 |
| 日期查询 | YYYY年MM月DD日 星期X | 今天是2026年03月19日 星期四 |
| 星期查询 | 星期X | 今天是星期四 |
| 月份查询 | MM月 | 现在是3月 |
| 年份查询 | YYYY年 | 今年是2026年 |
| 完整查询 | YYYY年MM月DD日 星期X HH:MM:SS | 当前日期时间是：2026年03月19日 星期四 21:50:15 |

### 语言偏好
- 默认使用中文格式
- 如果用户使用英文提问，可考虑使用英文格式
- 保持一致性：同一会话中使用相同语言格式

## 验证规则

### 格式验证
1. **日期验证**：
   - 月份必须在 1-12 之间
   - 日期必须符合月份的天数规则
   - 年份必须合理（如 1900-2100）

2. **时间验证**：
   - 小时必须在 0-23 之间
   - 分钟必须在 0-59 之间
   - 秒必须在 0-59 之间

3. **星期验证**：
   - 星期数字必须在 0-6 之间
   - 星期名称必须与日期匹配

### 错误处理
1. **格式错误**：返回默认格式的时间信息
2. **获取失败**：返回错误信息，建议用户检查系统时间
3. **意图不明确**：返回完整格式的时间信息
